A simple API wrapper, written in PHP, for the ValPal API, provided by Angels Media - https://valpal.co.uk.

Installation
-------

This package can be installed via [Composer]:

``` bash
$ composer require jedkirby/valpal-php
```

It requires **PHP >= 5.6.4**.
